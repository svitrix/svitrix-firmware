Nota: Para la primera instalacion, se recomienda borrar el dispositivo (ESP Web Tools ofrecera esta opcion). Esto garantiza una instalacion limpia.

Despues de la instalacion

  1. Conectate al punto de acceso WiFi El dispositivo creara una red svitrix_XXXXX con password 12345678. Conectate desde tu celular o computadora.
  2. Abre la interfaz web Navega a http://192.168.4.1 en tu navegador. Se abrira el panel de control de SVITRIX.
  3. Configura tu WiFi En la seccion de Settings, ingresa las credenciales de tu WiFi de casa. El dispositivo se reiniciara y se conectara a tu red.
  4. Listo! La IP aparecera en la pantalla LED. Abrela en el navegador para acceso completo. Configura el clima en http://[IP]/datafetcher

Configurar Clima (WeatherAPI)

Una vez conectado, ve a http://[IP]/datafetcher para agregar datos del clima:

  • Temperatura exterior
  • Sensacion termica
  • Humedad
  • Indice UV
  • Condicion del clima

Obtener API Key gratis en WeatherAPI.com
