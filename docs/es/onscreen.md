# Menú en Pantalla

SVITRIX proporciona un **menú en pantalla** directamente en tu reloj.
Mantén presionado el botón del medio por 2 segundos para acceder al menú.
Navega a través de los elementos con los botones izquierdo y derecho y elige el submenú presionando el botón del medio.
Mantén presionado el botón del medio por 2 segundos para salir del menú actual y guardar tu configuración.

::: warning
Puedes fácilmente encender o apagar tu matriz SVITRIX simplemente presionando dos veces el botón del medio si no estás en el Menú.
:::

| Elemento del Menú | Descripción |
| --- | --- |
| `BRIGHT` | Permite ajustar el brillo de la pantalla. Cambia entre control de brillo Automático y manual con el botón del medio. |
| `COLOR` | Permite seleccionar uno de 13 colores diferentes para el texto. Se muestran valores hexadecimales. |
| `SWITCH` | Determina si las páginas deben cambiar automáticamente. |
| `T-SPEED` | Ajusta la velocidad de transición entre apps. |
| `APPTIME` | Ajusta la duración de visualización de la app antes de cambiar a la siguiente. |
| `TIME` | Permite seleccionar el formato de hora. |
| `DATE` | Permite seleccionar el formato de fecha. |
| `WEEKDAY` | Permite seleccionar el inicio de semana. |
| `TEMP` | Permite seleccionar el sistema de temperatura (°C o °F). |
| `APPS` | Permite habilitar o deshabilitar apps internas. |
| `SOUND` | Permite habilitar o deshabilitar la salida de sonido. |
| `VOLUME` | Ajusta el volumen del buzzer (0–30). |
| `UPDATE` | Verifica y descarga nuevo firmware si está disponible. |
