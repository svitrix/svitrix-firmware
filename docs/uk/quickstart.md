
# Швидкий старт

Налаштування SVITRIX-XE1E займає лише кілька хвилин. Нижче — покроковий процес від розпакування до працюючого годинника.

## Крок 1. Прошивка пристрою

Підключіть Ulanzi TC001 до комп'ютера за допомогою USB-кабелю та відкрийте [онлайн-прошивальник](./flasher).

::: tip Порада
Якщо прошивка не запускається — спробуйте інший USB-кабель або порт. Деякі кабелі підтримують лише зарядку, але не передачу даних.
:::

## Крок 2. Підключення до WiFi

Після прошивки пристрій створить WiFi точку доступу:

| Параметр | Значення |
|----------|----------|
| Назва мережі | `svitrix_XXXXX` |
| Пароль | `12345678` |

Підключіться до цієї мережі зі смартфона або комп'ютера.

## Крок 3. Налаштування мережі

Відкрийте браузер і перейдіть за адресою:

**http://192.168.4.1**

З'явиться сторінка налаштування WiFi. Проскануйте доступні мережі, виберіть вашу, введіть пароль і натисніть Connect. Пристрій перезавантажиться і підключиться до вашої домашньої мережі.

## Крок 4. Знайдіть пристрій у мережі

Після підключення до WiFi пристрій покаже:
1. Версію прошивки (наприклад, `0.4.0`) протягом кількох секунд
2. Прокручуваний банер з "SVITRIX XE1E", IP-адресою та mDNS-ім'ям

Введіть IP-адресу у браузері, щоб відкрити [веб-інтерфейс](./webinterface).

::: tip Порада
Ви також можете використовувати `http://svitrix_XXXXX.local` (де XXXXX — ID вашого пристрою).
Якщо ви пропустили IP-адресу на екрані — знайдіть пристрій у списку підключених пристроїв вашого роутера.
:::

## Крок 5. Завантаження веб-інтерфейсу (тільки для розробників)

Якщо ви збираєте прошивку з вихідного коду, завантажте SPA веб-інтерфейс на пристрій:

```bash
cd web && npm run upload
```

::: tip Порада
Готові релізи прошивки включають цей крок в онлайн-прошивальнику. Це потрібно лише при збірці з вихідного коду.
:::

## Крок 6. Налаштування погоди (опціонально)

SVITRIX-XE1E включає застосунки погоди, які показують температуру, вологість, тиск та якість повітря на вулиці. Щоб увімкнути їх:

1. Зареєструйтесь на [WeatherAPI.com](https://www.weatherapi.com) (безкоштовно)
2. Скопіюйте ваш API-ключ з панелі керування
3. У веб-інтерфейсі перейдіть до **Settings → Weather**
4. Вставте ваш API-ключ
5. Виберіть метод визначення місцезнаходження:
   - **City Name** — наприклад, "Kyiv"
   - **Coordinates** — широта та довгота
   - **Auto (IP)** — автоматичне визначення за IP
   - **Station ID** — для персональних метеостанцій
6. Увімкніть потрібні застосунки погоди
7. Натисніть **Save Weather**

::: tip Порада
Безкоштовний план WeatherAPI.com дозволяє 1 мільйон запитів на місяць — більш ніж достатньо для оновлення кожні 10 хвилин.
:::

## Крок 7. Налаштування MQTT (опціонально)

У веб-інтерфейсі налаштуйте підключення до вашого MQTT-брокера. Це дозволить керувати годинником з HomeAssistant, IOBroker, NodeRed та інших систем розумного дому.

Детальніше про доступні команди — у розділі [API](./api).

## Готово!

SVITRIX-XE1E вже працює з попередньо встановленими застосунками: час, дата, температура, вологість, батарея та погода. Для створення власних застосунків ознайомтесь з розділом [Застосунки](./apps).
