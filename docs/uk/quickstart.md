
# Швидкий старт

Налаштування SVITRIX займає лише кілька хвилин. Нижче — покроковий процес від розпакування до працюючого годинника.

## Крок 1. Прошивка пристрою

Підключіть Ulanzi TC001 до комп'ютера за допомогою USB-кабелю та відкрийте [онлайн-прошивальник](./flasher).

::: tip Порада
Якщо прошивка не запускається — спробуйте інший USB-кабель або порт. Деякі кабелі підтримують лише зарядку, але не передачу даних.
:::

## Крок 2. Підключення до WiFi

Після прошивки пристрій створить WiFi точку доступу:

| Параметр | Значення |
|----------|----------|
| Назва мережі | `svitrix_XXXXX` |
| Пароль | `12345678` |

Підключіться до цієї мережі зі смартфона або комп'ютера.

## Крок 3. Налаштування мережі

Відкрийте браузер і перейдіть за адресою:

**http://192.168.4.1**

З'явиться сторінка налаштування WiFi. Проскануйте доступні мережі, виберіть вашу, введіть пароль і натисніть Connect. Пристрій перезавантажиться і підключиться до вашої домашньої мережі.

## Крок 4. Знайдіть пристрій у мережі

Після підключення до WiFi на матриці з'явиться IP-адреса пристрою. Введіть цю адресу у браузері, щоб відкрити [веб-інтерфейс](./webinterface).

::: tip Порада
Якщо ви пропустили IP-адресу на екрані — знайдіть пристрій у списку підключених пристроїв вашого роутера.
:::

## Крок 5. Завантаження веб-інтерфейсу (тільки для розробників)

Якщо ви збираєте прошивку з вихідного коду, завантажте SPA веб-інтерфейс на пристрій:

```bash
cd web && npm run upload
```

::: tip Порада
Готові релізи прошивки включають цей крок в онлайн-прошивальнику. Це потрібно лише при збірці з вихідного коду.
:::

## Крок 6. Налаштування MQTT

У веб-інтерфейсі налаштуйте підключення до вашого MQTT-брокера. Це дозволить керувати годинником з HomeAssistant, IOBroker, NodeRed та інших систем розумного дому.

Детальніше про доступні команди — у розділі [API](./api).

## Готово!

SVITRIX вже працює з попередньо встановленими застосунками: час, дата, температура та вологість. Для створення власних застосунків ознайомтесь з розділом [Застосунки](./apps).
