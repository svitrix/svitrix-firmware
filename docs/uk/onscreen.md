# Екранне меню

SVITRIX має **екранне меню** безпосередньо на годиннику.
Натисніть і утримуйте середню кнопку 2 секунди для доступу до меню.
Переміщуйтесь між пунктами лівою та правою кнопками, а вибір підменю — натисканням середньої кнопки.
Утримуйте середню кнопку 2 секунди, щоб вийти з поточного меню та зберегти налаштування.

::: warning
Ви можете легко увімкнути або вимкнути матрицю SVITRIX, просто двічі натиснувши середню кнопку, якщо ви не в меню.
:::

| Пункт меню | Опис |
| --- | --- |
| `BRIGHT` | Регулювання яскравості дисплея. Перемикання між автоматичним і ручним режимом — середньою кнопкою. |
| `COLOR` | Вибір одного з 13 кольорів тексту. Відображаються HEX-значення. |
| `SWITCH` | Визначає, чи сторінки повинні автоматично перемикатися. |
| `T-SPEED` | Регулювання швидкості переходу між застосунками. |
| `APPTIME` | Регулювання тривалості відображення застосунку перед перемиканням. |
| `TIME` | Вибір формату часу. |
| `DATE` | Вибір формату дати. |
| `WEEKDAY` | Вибір першого дня тижня. |
| `TEMP` | Вибір системи температури (°C або °F). |
| `APPS` | Увімкнення або вимкнення внутрішніх застосунків. |
| `SOUND` | Увімкнення або вимкнення звукового виходу. |
| `VOLUME` | Регулювання гучності зумера (0–30). |
| `UPDATE` | Перевірка та завантаження нової прошивки, якщо доступна. |
