# Резервне копіювання

Функціонал резервного копіювання та відновлення доступний у [веб-інтерфейсі](./webinterface) на сторінці **Backup** (`/backup`).

## Резервне копіювання (завантаження)

Натисніть **Download Backup**, щоб зберегти всі файли з файлової системи пристрою (іконки, мелодії, палітри, користувацькі додатки, конфігурацію) в один JSON-файл.

## Відновлення (завантаження)

Виберіть раніше завантажений файл бекапу для відновлення. Усі файли будуть завантажені на пристрій, і він автоматично перезавантажиться після успішного відновлення.

::: warning Увага
Відновлення бекапу перезапише існуючі файли на пристрої. Переконайтеся, що відновлюєте правильний бекап.
:::
