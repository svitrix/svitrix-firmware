# Звуки

SVITRIX використовує вбудований пасивний зумер у годиннику Ulanzi TC001 для відтворення монофонічних мелодій RTTTL.

## Відтворення мелодій

Ви можете відтворювати мелодії RTTTL двома способами:

**Через API:**
Надішліть рядок RTTTL безпосередньо з вашим API-запитом.
Дивіться документацію команд:
- [Окремі звуки](./api#sound-playback)
- [З вашим сповіщенням](./api#json-properties)

**Через файл:**
Цей метод дозволяє уникнути довгих JSON, оскільки буфер прийому обмежений.
Перейдіть до файлового менеджера у [веб-інтерфейсі](./webinterface) та створіть новий текстовий файл у папці `MELODIES`.
Назвіть його як завгодно, але з розширенням `.txt`, наприклад `alarm.txt`. Всередині файлу розмістіть мелодію у форматі RTTTL.
При використанні звукового файлу в API-запитах опускайте розширення.

Ви можете знайти багато мелодій в інтернеті:
* [Laub-Home Wiki: RTTTL Songs](https://www.laub-home.de/wiki/RTTTL_Songs)
* [Online RTTTL player](https://adamonsoon.github.io/rtttl-play/)
* [RTTTL editor](https://corax89.github.io/esp8266Game/soundEditor.html)

## Регулювання гучності

Гучність контролює шпаруватість ШІМ-сигналу зумера. Діапазон: **0–30** (за замовчуванням: **25**).

**Екранне меню:**
Перейдіть до пункту `VOLUME` на годиннику. Використовуйте кнопки ліворуч/праворуч для налаштування. Довге натискання кнопки вибору для збереження.

**Через API:**
```bash
# Встановити гучність на 15
curl -X POST http://<ip>/api/settings -d '{"VOL": 15}'

# Отримати поточну гучність
curl http://<ip>/api/settings
```

Налаштування гучності зберігається у флеш-пам'яті та не скидається при перезавантаженні.

## Увімкнення/вимкнення звуку

Звук можна увімкнути або вимкнути глобально:

**Екранне меню:**
Перейдіть до пункту `SOUND`. Перемикайте кнопками ліворуч/праворуч. Довге натискання кнопки вибору для збереження.

**Через API:**
```json
{"SOUND": true}
```

Коли звук вимкнено, все відтворення мелодій (сповіщення, звук завантаження, API-виклики) заглушується. Рівень гучності зберігається та відновлюється при повторному увімкненні звуку.
