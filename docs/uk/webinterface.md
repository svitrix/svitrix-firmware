# Веб-інтерфейс

Після підключення SVITRIX до вашої Wi-Fi мережі ви можете отримати доступ до веб-інтерфейсу через порт 80. Для цього просто введіть IP-адресу SVITRIX у вашому браузері. Ця IP-адреса відображається на матриці при кожному завантаженні.
Веб-інтерфейс надає основні налаштування, необхідні для початкового налаштування:

| Розділ       | Опис                                                                                                                                                                                         |
|--------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **WiFi Setup** | Налаштування Wi-Fi та зміна точки доступу за потреби.                                                                                                                                        |
| **Network**    | Призначення статичної IP-адреси для SVITRIX.                                                                                                                                                 |
| **MQTT**       | Налаштування MQTT брокера та увімкнення виявлення Home Assistant, якщо ви використовуєте цю систему розумного дому.                                                                           |
| **Time**       | Налаштування сервера часу та часового поясу для коректного відображення часу та дати. Також підтримує автоматичний перехід на літній/зимовий час.                                               |
| **Icons**      | Використання іконок з LaMetric. SVITRIX має вбудований завантажувач. Введіть ID іконки, і SVITRIX завантажить її для негайного використання.                                                   |
| **Auth**       | Захист веб-інтерфейсу та API базовою автентифікацією. Кожна сторінка, API-запит і застосунок SVITRIX вимагатимуть ці облікові дані. Не втрачайте їх, інакше доведеться повністю скинути SVITRIX. |
| **Files**      | Доступ до вбудованого файлового менеджера для завантаження даних, таких як іконки або мелодії RTTTL.                                                                                          |
| **Update**     | Ручне завантаження оновлення у розділі "Firmware". Необхідний .bin файл можна знайти у [розділі релізів SVITRIX](https://github.com/svitrix/svitrix-firmware/releases).                             |
| **LiveView**   | Перегляд матриці SVITRIX у реальному часі, створення скріншотів та анімованих GIF.                                                                                                           |
| **Backup**     | Система резервного копіювання компілює всю флеш-пам'ять у zip-файл для завантаження. Ви можете відновити цю резервну копію на іншому пристрої SVITRIX.                                        |
| **Docs**       | Посилання на офіційну документацію SVITRIX.                                                                                                                                                  |
| **Flows**      | Посилання на вебсайт SVITRIX Flows.                                                                                                                                                          |
