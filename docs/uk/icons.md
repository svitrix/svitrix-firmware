# Іконки

Щоб персоналізувати ваш застосунок або сповіщення улюбленою іконкою, ви можете скористатися одним із цих простих методів:

### Завантаження іконки

1. **Вбудований завантажувач**: Відкрийте веб-інтерфейс і перейдіть на вкладку іконок.
2. **Введіть ID іконки**: Вкажіть ID іконки LaMetric у відповідному полі.
3. **Попередній перегляд**: Переконайтеся, що це потрібна іконка.
4. **Завантаження**: Натисніть, щоб завантажити іконку та використовувати її негайно без перезавантаження чи конвертації.

Ви можете переглянути та знайти іконки LaMetric на [LaMetric Icon Gallery](https://developer.lametric.com/icons).

### Застосунок SVITRIX

За допомогою застосунку SVITRIX, доступного для Android та iOS, ви отримуєте ексклюзивний доступ до офіційної бази даних SVITRIX з понад 1800 іконками та анімаціями у форматах 8x8 та 32x8. Застосунок також дозволяє створювати власні іконки та завантажувати їх безпосередньо на пристрій SVITRIX або у хмарну базу даних.

### Власні іконки

Ви можете створювати власні іконки за допомогою будь-якого графічного редактора та завантажувати їх через файловий менеджер веб-інтерфейсу до папки "ICONS". Переконайтеся, що іконки відповідають наступним специфікаціям:

- **GIF**: Максимальний розмір 32x8 пікселів.
- **JPG**: Максимальний розмір 8x8 пікселів.

### Важлива примітка

Рендерер GIF у SVITRIX наразі підтримує лише 8-бітні GIF без прозорості. Якщо ви помітили графічні артефакти на матриці, спробуйте замінити прозорі пікселі на суцільний чорний за допомогою онлайн-інструменту, наприклад [Online GIF Tools](https://onlinegiftools.com/add-gif-background).
